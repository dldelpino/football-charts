# Football Charts

This project aims to collect and display football data related to the top 5 European leagues since the 2000/01 season. It has been coded entirely in Python, using [NiceGUI](nicegui.io) for the user interface and [Highcharts](highcharts.com) for pie charts and bar charts. Data is stored in CSV files which have been generated by scraping [FBref](fbref.com). Team badge icons have been taken from [Transfermarkt](transfermarkt.com).
